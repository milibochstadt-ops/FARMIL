# Farmil

## Descripción

Farmil es un sistema web desarrollado para administrar una farmacia. Permite gestionar medicamentos, usuarios, ventas y apertura de caja.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- XAMPP

## Instalación

1. Descargar o clonar este repositorio.
2. Copiar la carpeta del proyecto en `htdocs` de XAMPP.
3. Iniciar Apache y MySQL desde XAMPP.
4. Importar la base de datos en phpMyAdmin.
5. Abrir el proyecto desde:

```
http://farmacia-mili.gt.tc/login.php
```
